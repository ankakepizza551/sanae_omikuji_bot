# ⛩️ おみくじ早苗BOT (Sanae Omikuji Bot)

東風谷早苗がDiscordサーバーで守矢神社の信仰を集めるための、東方Project二次創作Discord Botです。
毎日のおみくじやお賽銭、奇跡への挑戦を通じて、早苗との「信仰度（好感度）」を高めていくことができます。

---

## 🌟 主な機能とコマンド

本Botは、通常のテキストコマンド（デフォルトプレフィックス `!`）および **スラッシュコマンド（`/`）** の両方に対応しています（ハイブリッドコマンド）。

### 1. 守矢神社おみくじ (`/omikuji` または `!omikuji`)
1日1回限定で引くことができるおみくじです。
おみくじを引くと、参拝者の名前・日付・運勢・早苗の託宣メッセージ・ラッキーアイテム/アクション・信仰度ゲージがデザインされた **「特製おみくじ画像」が自動生成され、Discord上に送信** されます。

- **運勢の種類:** 奇跡、常識無視、大吉、中吉、吉、小吉、末吉、凶、大凶
- 運勢に応じて獲得できる信仰度（好感度）や内容が異なります。毎日引くことで連続参拝日数がカウントされます。

### 2. お賽銭の奉納 (`/moriya [金額]` または `!moriya [金額]`)
1日1回限定で、守矢神社のお賽銭箱にコイン（数値）を奉納します。
投げ入れた金額によって早苗の反応メッセージが4段階に変化し、獲得できる信仰度もアップします。
- 10コイン未満: 信仰度 `+1`
- 10〜99コイン: 信仰度 `+5`
- 100〜999コイン: 信仰度 `+12`
- 1000コイン以上: 信仰度 `+25` (博麗神社との対比リアクションなど)

### 3. 奇跡への挑戦 (`/miracle` または `!miracle`)
1日1回、早苗に祈りを捧げて奇跡を起こすミニゲームに挑戦します。
- **成功率:** 初期確率は **20%** ですが、現在の信仰度に応じて確率が上昇します（信仰度1につき `+0.05%` 上昇、最大 **80%**）。
- **結果:** 成功すると信仰度が大きく上昇（`+50`）し、特別な演出メッセージが表示されます。失敗しても少しだけ信仰度が上がります（`+2`）。

### 4. 参拝手帳（プロフィール）の確認 (`/profile` または `!profile`)
これまでの参拝状況や、早苗からの評価ステータスを確認できます。
- **信仰ステータス:** 信仰度（0〜1000+）に応じた称号（「一般参拝客」から「早苗の親友（常識を捨てし者）」まで7段階）、好感度ゲージ、連続参拝日数を表示。
- **参拝記録:** 累計おみくじ回数、累計お賽銭額。
- **おみくじの内訳:** どの運勢を何回引いたかの統計。
- **早苗からの言葉:** 信仰度のランクに応じて、早苗からのメッセージが変化します。

## 🛠️ 管理者（オーナー）専用コマンド

Botのオーナーのみが実行できる管理用コマンドです。

- **`!sync`**: スラッシュコマンドをDiscordサーバーに同期します。Bot導入後に一度実行してください。
- **`!reset_cooldown [メンバー]`**: 指定したメンバー（未指定なら自分）のおみくじ、お賽銭、奇跡のクールダウン（1日1回制限）を即座にリセットします。
- **`!bot_status`**: 接続サーバー数、レイテンシ（応答速度）、稼働中のモジュール（Cogs）のリストを表示します。

## 🚀 導入と起動方法

### 1. 前提要件
- Python 3.8 以上
- Discord Botのアカウント（Token）およびメッセージインテント（Message Content Intent）の有効化

### 2. インストール
必要なパッケージをインストールします。
```bash
pip install -r requirements.txt
```

### 3. 環境設定
`.env.example` をコピーして `.env` ファイルを作成し、各種設定値を入力します。
```env
DISCORD_TOKEN=ここにDiscord_Botのトークンを記入
BOT_PREFIX=!
DB_PATH=data/sanae_omikuji.db
```

### 4. 起動
`main.py` を実行してBotを起動します。
```bash
python main.py
```
起動すると、自動的にデータベース（SQLite）の初期化が行われ、Botのステータスに「守矢神社でお守り配り | !omikuji」と表示されます。
